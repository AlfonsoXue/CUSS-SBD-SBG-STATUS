# 機場自助化設備建置儀表板 (CUSS / SBD / SBG Status Dashboard)

長榮航空全球場站「旅客自助旅程」三階段自助化設備建置狀態的可互動儀表板。
單一 HTML 檔、零相依、離線可用，適合放入簡報或隨處分享。

## 開啟方式

直接用瀏覽器開啟 [`index.html`](./index.html) 即可（雙擊或拖入瀏覽器），無需安裝或建置。
所有編輯會自動存於瀏覽器 `localStorage`，重開仍在。

## 三階段設備

| 階段 | 說明 |
|------|------|
| **CUSS** | 自助報到 / 列印登機證 |
| **SBD**  | 自助行李託運 |
| **SBG**  | 自助登機門 / 生物辨識 |

## 場站現況（5 種狀態）

| 記號 | 狀態 | 色彩 |
|------|------|------|
| ● | 已上線 | EVA Green `#00853C` |
| ○ | 已上線暫關 | `#7FA893` |
| ▲ | 建置進行中 | `#E8A400` |
| △ | 評估中 | `#C2AC7E` |
| ✕ | 無設備 | `#DCDBD6` |

## 功能

- **左欄** — 三階段覆蓋率長條與圖例，依目前納管場站即時計算百分比與站數。
- **右欄** — 場站方塊：上緣＝CUSS 現況、下緣＝SBD 現況、右上白點＝2026 經理人 KPI 場站、方塊內顯示系統商。
  - **依時程／依區域** 兩種分組檢視可切換。
  - **游標移至方塊** 顯示完整資訊（現況／系統商／啟用時程／各站進度）。
- **編輯面板** — 點方塊開啟，可調整：場站代碼、區域、建置時程、CUSS／SBD／SBG 現況、各設備**系統商**與**啟用／時程**、KPI 標記、各站進度、排程備註、刪除場站。
- **快速切換模式** — 開啟後點方塊上半循環切換 CUSS、下半切換 SBD，快速批次更新。
- **新增場站**、**重設**（還原原始資料庫）。
- **匯出** — PNG（放簡報用）、JSON、CSV（含所有細項欄位，UTF-8 BOM 相容 Excel）。

## 資料來源

初始資料整合自兩份原始資料庫（資料基準 2026·05）：

- **全球場站自動化綜整（202605）** — 57 個納管場站的 CUSS／SBD／SBG 現況、系統商、各站進度。
- **2026 CUSS 排程** — 各站建置時程分期與啟用日期。

共 57 站，CUSS 已上線 18（32%）、SBD 9（16%）、SBG 8（14%）、KPI 場站 10。
所有數字會隨編輯即時重算。

## 技術

原生 HTML／CSS／JavaScript，單一檔案，無外部相依。
唯一的線上資源是「匯出圖片」時才動態載入的 `html2canvas`；離線時可改用匯出 JSON／CSV。
配色採用 EVA Air 設計系統色票。
