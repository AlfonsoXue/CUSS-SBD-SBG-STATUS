# 機場自助化設備建置儀表板 (CUSS / SBD / SBG Status Dashboard)

長榮航空全球場站「旅客自助旅程」三階段自助化設備建置狀態的可互動儀表板。
單一 HTML 檔、零相依、離線可用，適合放入簡報或隨處分享。

## 開啟方式

直接用瀏覽器開啟 [`index.html`](./index.html) 即可（雙擊或拖入瀏覽器），無需安裝或建置。
所有編輯會自動存於瀏覽器 `localStorage`，重開仍在。

## 三階段設備

| 階段 | 說明 |
|------|------|
| **CUSS** | 自助報到 / 列印登機證 |
| **SBD**  | 自助行李託運 |
| **SBG**  | 自助登機門 / 生物辨識 |

## 場站現況（5 種狀態）

| 記號 | 狀態 | 色彩 |
|------|------|------|
| ● | 已上線 | EVA Green `#00853C` |
| ○ | 已上線暫關 | `#7FA893` |
| ▲ | 建置進行中 | `#E8A400` |
| △ | 評估中 | `#C2AC7E` |
| ✕ | 無設備 | `#DCDBD6` |

頂端可切換兩種檢視：**時程儀表板** 與 **地圖 · 定位矩陣**。

### 時程儀表板

- **左欄** — CUSS｜SBD｜SBG 覆蓋率長條與圖例，依目前納管場站即時計算百分比與站數。
  - **設備搭配篩選**（覆蓋率卡右上下拉複選，最多 3 項）：勾選 CUSS／SBD／SBG 的已上線搭配（皆有、CUSS+SBD、CUSS+SBG…）後，左欄覆蓋率卡即改為顯示每個所選搭配一張卡，**卡片標題隨勾選變更**為對應搭配名稱（CUSS+SBD+SBG、CUSS+SBD、CUSS+SBG…），並顯示該搭配占全部場站的比率與符合／其餘站數長條。多數搭配採「包含」計算（列出的階段皆已上線即計入，例如 CUSS+SBD 亦含三階段皆有的場站）；**僅CUSS+SBG** 例外採「不包含」計算（CUSS 與 SBG 已上線但 SBD 未上線），故標題冠以「僅」以利辨識。單獨勾選 CUSS／SBD／SBG 之一時，該卡沿用原本各狀態（已上線／暫關／建置中／評估中／無設備）堆疊長條。未勾選時則顯示預設的 CUSS／SBD／SBG 三張覆蓋率卡。每張覆蓋率卡下方會**列出對應場站代碼**（符合／已上線場站）；**游標移至卡片時**，右側對應場站方塊會自動以橘色外框圈起、卡片本身亦高亮，移開即消失，方便快速辨識。
- **右欄** — 場站方塊：上緣＝CUSS 現況、下緣＝SBD 現況、右上白點＝2026 經理人 KPI 場站、方塊內顯示系統商。
  - **CUSS 列印行李條功能** 開關：標記後方塊左上角顯示行李條示意圖（初始值依各站進度自動判定）。
  - 群組內方塊排序優先序：① 三階段完備＋可列印行李條 ② CUSS+SBD 完備＋可列印行李條 ③ 三階段完備 ④ CUSS/SBD 完備 ⑤ 其餘依自動化完備度高→低。**依區域檢視**時，先依建置時程（排程）排序，再套用上述完備度規則。
  - **依時程／依區域** 兩種分組檢視可切換。
  - **顯示設定**（右欄工具列）：**場站名稱字級** 滑桿（方塊內系統商文字同步等比縮放）與 **圖例字級** 滑桿（圖例色塊同步等比縮放）可即時調整；**圖例圖形（實心圓／三角）** 開關可決定圖例是否顯示 ●▲△○✕ 符號，色塊與文字則恆常顯示；此開關同時連動更新左欄 CUSS｜SBD｜SBG 覆蓋率各卡內的圖例符號。
  - **游標移至方塊** 顯示完整資訊（現況／系統商／啟用時程／各站進度）。

### 地圖 · 定位矩陣

- **全球場站分布圖** — 各場站依實際經緯度標於世界地圖；圓點大小＝旅客量、白框＝KPI。
  - **圓點著色**：**現況（上半＝CUSS／下半＝SBD 兩色）** 或 **所屬區域**（單色）。顯示名稱時 CUSS 上半比例自動加大以容納文字。
  - **場站名稱** 可顯示／隱藏（顯示於圓點上）；文字互相遮蔽時會自動拉出並以細線條引導，縮放時重新排列以維持可讀性。
  - 圓點會依地理位置自動分散排列，避免互相重疊（密集區如東北亞會擴散呈現）。
  - **地圖變色**：深／淺色切換；區域配色可選 **彩色分區／灰階／品牌綠／依覆蓋率**（依各洲 CUSS 上線比例做綠色深淺 choropleth）。**場站名稱** 與 **KPI 白框** 以開關（switch）切換顯示，並可用滑桿調整 **圖例大小**。
  - **拖曳移動、滾輪縮放**；篩選列右側整合 ＋／－ 縮放、重置視圖、**圓圈大小** 與 **場站名稱字級** 調整（拖曳字級滑桿會自動開啟場站名稱顯示），並提供 **恢復預設** 按鈕一鍵還原地圖縮放／圓圈大小／名稱字級。**清除篩選** 按鈕固定於篩選列左側（無啟用篩選時呈灰階停用）。
- **場站策略定位矩陣** — 2×2 象限，Y 軸＝自動化完備度（CUSS／SBD／SBG 加權）、X 軸＝旅客量／航班量（對數尺度）。四象限：
  - **成熟場站**（高完備・低量）、**重點場站**（高完備・高量）、**進行中場站**（低完備・高量）、**限制型場站**（低完備・低量）。
  - 各場站以圓點落於對應象限（上半＝CUSS／下半＝SBD 兩色），依位置整齊排列於網格、互不重疊，隨編輯即時重算；每個象限並以大型圖示浮水印標示。
  - **X 軸依當前顯示（篩選後）場站的量體自動重新縮放** — 例如排除 TPE（超大量體）後，其餘場站會依比例重新分布於矩陣中。
  - 四象限格內只保留圖示，文字說明移至矩陣下方圖例，避免被場站圓點遮擋。
- **篩選** — 地圖與矩陣共用篩選列，可依 **場站區域／CUSS／SBD／SBG／經理人 KPI** 篩選，並提供 **個別場站** 選取（逐站加入／排除），即時反映站數與矩陣縮放。
- **分開匯出圖片** — 地圖與定位矩陣各有獨立「匯出圖片」按鈕，另保留整體匯出。

### 共用功能

- **編輯面板** — 點方塊／圓點開啟，可調整：場站代碼、區域、建置時程、CUSS／SBD／SBG 現況、各設備**系統商**與**啟用／時程**、KPI 標記、**旅客量／航班量**、各站進度、排程備註、刪除場站。
- **快速切換模式** — 開啟後點方塊上半循環切換 CUSS、下半切換 SBD，快速批次更新。
- **新增場站**、**重設**（還原原始資料庫）。
- **匯出** — PNG（高解析度，約 4× 裝置像素，並自動限制在瀏覽器最大畫布尺寸內）、JSON、CSV（含系統商、旅客量／航班量、自動化完備度、定位象限等所有細項，UTF-8 BOM 相容 Excel）。

## 資料來源

初始資料整合自三份原始資料庫（資料基準 2026·05）：

- **全球場站自動化綜整（202605）** — 57 個納管場站的 CUSS／SBD／SBG 現況、系統商、各站進度。
- **2026 CUSS 排程** — 各站建置時程分期與啟用日期。
- **場站航班量／旅客量** — 定位矩陣 X 軸與地圖圓點大小之依據。

共 57 站，CUSS 已上線 18（32%）、SBD 9（16%）、SBG 8（14%）、KPI 場站 10。
所有數字會隨編輯即時重算。世界地圖幾何取自 Natural Earth（110m，公有領域），已簡化並內嵌。

## 技術

原生 HTML／CSS／JavaScript，單一檔案，無外部相依。
唯一的線上資源是「匯出圖片」時才動態載入的 `html2canvas`；離線時可改用匯出 JSON／CSV。
配色採用 EVA Air 設計系統色票。
