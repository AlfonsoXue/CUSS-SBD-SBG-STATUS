# 機場自助化設備建置儀表板 (CUSS / SBD / SBG Status Dashboard)

長榮航空全球場站「旅客自助旅程」三階段自助化設備建置狀態的可互動儀表板。
單一 HTML 檔、零相依、離線可用，適合放入簡報或隨處分享。

## 開啟方式

直接用瀏覽器開啟 [`index.html`](./index.html) 即可（雙擊或拖入瀏覽器），無需安裝或建置。
所有編輯會自動存於瀏覽器 `localStorage`，重開仍在。

## 三階段設備

| 階段 | 說明 |
|------|------|
| **CUSS** | 自助報到 / 列印登機證 |
| **SBD**  | 自助行李託運 |
| **SBG**  | 自助登機門 / 生物辨識 |

## 場站現況（5 種狀態）

| 記號 | 狀態 | 色彩 |
|------|------|------|
| ● | 已上線 | EVA Green `#00853C` |
| ○ | 已上線暫關 | `#7FA893` |
| ▲ | 建置進行中 | `#E8A400` |
| △ | 評估中 | `#C2AC7E` |
| ✕ | 無設備 | `#DCDBD6` |

左上角為 **EVA AIR 長榮航空** 官方企業識別標誌（官方原圖以 base64 內嵌 PNG，維持單檔自足），其後為標題「機場自助化設備建置儀表板」。頂端可切換兩種檢視：**時程儀表板** 與 **地圖 · 定位矩陣**。

### 時程儀表板

- **左欄** — CUSS｜SBD｜SBG 覆蓋率長條與圖例，依目前納管場站即時計算百分比與站數。
  - **設備搭配篩選**（覆蓋率卡右上下拉複選，最多 3 項）：勾選 CUSS／SBD／SBG 的已上線搭配（皆有、CUSS+SBD、CUSS+SBG…）後，左欄覆蓋率卡即改為顯示每個所選搭配一張卡，**卡片標題隨勾選變更**為對應搭配名稱（CUSS+SBD+SBG、CUSS+SBD、CUSS+SBG…），並顯示該搭配占全部場站的比率與符合／其餘站數長條。多數搭配採「包含」計算（列出的階段皆已上線即計入，例如 CUSS+SBD 亦含三階段皆有的場站）；**僅CUSS+SBG** 例外採「不包含」計算（CUSS 與 SBG 已上線但 SBD 未上線），故標題冠以「僅」以利辨識。單獨勾選 CUSS／SBD／SBG 之一時，該卡沿用原本各狀態（已上線／暫關／建置中／評估中／無設備）堆疊長條。未勾選時則顯示預設的 CUSS／SBD／SBG 三張覆蓋率卡。每張覆蓋率卡下方會**列出對應場站代碼**（符合／已上線場站）；**游標移至卡片時**，右側對應場站方塊會自動以橘色外框圈起、卡片本身亦高亮，移開即消失，方便快速辨識。
- **右欄** — 場站方塊：上緣＝CUSS 現況、下緣＝SBD 現況、右上白點＝2026 經理人KPI場站、方塊內顯示系統商。
  - **CUSS 列印行李條功能** 開關：標記後方塊左上角顯示行李條示意圖（初始值依各站進度自動判定）。
  - 群組內方塊排序優先序：① 三階段完備＋可列印行李條 ② CUSS+SBD 完備＋可列印行李條 ③ 三階段完備 ④ CUSS/SBD 完備 ⑤ 其餘依自動化完備度高→低。**依區域檢視**時，先依建置時程（排程）排序，再套用上述完備度規則。
  - **依時程／依區域** 兩種分組檢視可切換。
  - **顯示設定**（右欄工具列）：**場站名稱字級** 滑桿（方塊內系統商文字同步等比縮放）與 **圖例字級** 滑桿（圖例色塊同步等比縮放）可即時調整；**圖例圖形（實心圓／三角）** 開關可決定圖例是否顯示 ●▲△○✕ 符號，色塊與文字則恆常顯示（同時連動左欄覆蓋率各卡內的圖例符號）；**經理人 KPI** 開關（switch）可顯示／隱藏方塊上的 KPI 白點；**回復預設** 按鈕一鍵還原上述所有顯示設定。
  - **游標移至方塊** 顯示完整資訊（現況／系統商／啟用時程／各站進度）。

### 地圖 · 定位矩陣

- **全球場站自動化設備概況** — 各場站依實際經緯度標於世界地圖；圓點大小＝旅客量、白框＝經理人KPI。
  - **圓點著色**：**現況（上半＝CUSS／下半＝SBD 兩色）** 或 **所屬區域**（單色）。顯示名稱時 CUSS 上半比例自動加大以容納文字。
  - **場站名稱** 可顯示／隱藏（顯示於圓點上）；文字互相遮蔽時會自動拉出並以細線條引導，縮放時重新排列以維持可讀性。
  - 圓點會依地理位置自動分散排列，避免互相重疊（密集區如東北亞會擴散呈現）。
  - **自動聚焦（不壓縮、不失真）**：地圖**維持真實等距投影，洲界比例不變**，僅以**均勻裁切**自動將視窗框住所有場站圓點（含半徑與名稱標籤空間），確保**所有圓圈完整顯示於窗格內且更大**；重新聚焦時以**簡易摺疊／展開動畫**呈現，不擠壓地圖。篩選後亦自動重新聚焦於當前可見場站。手動縮放／拖曳後可用「重置視圖／恢復預設」回到自動聚焦。
  - **地圖變色**：深／淺色切換；區域配色可選 **彩色分區／灰階／品牌綠／依覆蓋率**（依各洲 CUSS 上線比例做綠色深淺 choropleth）。**場站名稱** 與 **KPI 白框** 以開關（switch）切換顯示，圖例列右側並可用滑桿調整 **圖例大小**。
  - **拖曳移動、滾輪縮放**；縮放時視窗會自動**貼齊完整圓圈**（圓心落於視窗內者一律完整顯示，不被窗格邊緣切半），拖曳／縮放亦**限制於場站範圍內**（不會滑出空白海域，也不會縮小到超過自動聚焦範圍），確保放大後圓圈不跑出窗格外；**單獨檢視部分板塊**（如僅篩選一個區域）時，視窗框架會依內容比例自動收攏（板塊比例不變、不失真），讓板塊**盡量填滿視窗**。「全球場站自動化設備概況」面板標題右側（匯出按鈕之前）整合 ＋／－ 縮放（每次 **0.1**、範圍 ×1.0–**×10.0**；一開始放大（×1.1），**未移動的板塊即完全灰階**並隨縮放逐步淡出，移動中的板塊維持原色）、重置視圖、**圓圈大小** 與 **場站名稱字級** 調整（兩者最小 **5%**）（拖曳字級滑桿會自動開啟場站名稱顯示），並提供 **恢復預設** 按鈕一鍵還原地圖縮放／圓圈大小／名稱字級。**清除篩選** 按鈕固定於篩選列左側（無啟用篩選時呈灰階停用）。
- **場站策略定位矩陣** — 2×2 象限，Y 軸＝自動化完備度（CUSS／SBD／SBG 加權）、X 軸＝旅客量／航班量（對數尺度）。四象限：
  - **成熟場站**（高完備・低量）、**重點場站**（高完備・高量）、**進行中場站**（低完備・高量）、**限制型場站**（低完備・低量）。
  - 各場站以圓點落於對應象限（上半＝CUSS／下半＝SBD 兩色），依位置整齊排列於網格、互不重疊，隨編輯即時重算；每個象限並以大型圖示浮水印標示。
  - **X 軸依當前顯示（篩選後）場站的量體自動重新縮放** — 例如排除 TPE（超大量體）後，其餘場站會依比例重新分布於矩陣中。
  - 四象限格內只保留圖示，文字說明移至矩陣下方圖例，避免被場站圓點遮擋。
- **篩選** — 地圖與矩陣共用篩選列，可依 **場站區域／CUSS／SBD／SBG／經理人 KPI** 篩選，並提供 **個別場站** 選取（逐站加入／排除），即時反映站數與矩陣縮放。個別場站清單於**點擊空白處或按 Esc 時自動收合**（點選場站時保持展開）。
- **分開匯出圖片** — 地圖與定位矩陣各有獨立「匯出圖片」按鈕，另保留整體匯出。

### 共用功能

- **編輯面板** — 點方塊／圓點開啟，可調整：場站代碼、區域、建置時程、CUSS／SBD／SBG 現況、各設備**系統商**與**啟用／時程**、KPI 標記、**旅客量／航班量**、各站進度、排程備註、刪除場站、**隱藏場站**。
- **隱藏場站** — 於編輯面板點「隱藏場站」後，該站**不顯示於任何檢視**（時程／區域／地圖／矩陣），也**不列入**覆蓋率、地圖與矩陣計算與站數。標題列出現「已隱藏 N」按鈕，開啟管理視窗可逐一或全部「取消隱藏」。
- **資料最後更新時間** — 頁面右下角顯示資料最後更新時間（`DDMMM HH:MM`），每次編輯／匯入／隱藏等異動即更新。
- **快速切換模式** — 開啟後點方塊上半循環切換 CUSS、下半切換 SBD，快速批次更新。
- **新增場站**、**重設**（還原為「本次開啟檔案時、尚未更新前」的狀態基準點，而非原始資料庫；若要回到最初 57 站原始資料庫，請改用匯入或重新載入未編輯來源）。
- **匯出** — PNG（高解析度，約 4× 裝置像素，並自動限制在瀏覽器最大畫布尺寸內）；**匯出資料**（標題列按鈕，滑鼠移過即彈出選單）可選 JSON 或 CSV（含系統商、旅客量／航班量、自動化完備度、定位象限等所有細項，UTF-8 BOM 相容 Excel）。標題列功能鈕於足夠寬度時單列顯示於標題右側。
- **另存新檔（HTML 快照）** — 一鍵下載「可獨立開啟」的自足 HTML 檔，內含**當前資料與更新紀錄**；預設檔名為 `機場自助化設備建置儀表板(最後更新:DDMMM HH:MM).html`。開啟該檔即顯示存檔當下的資料，適合封存或分享（使用獨立儲存鍵，不影響主儀表板）。
- **更新紀錄日誌** — 標題列「更新紀錄」開啟日誌視窗，記錄每次編輯欄位、新增／刪除場站、匯入資料、重設與另存 HTML 的時間與內容（由新到舊，最多 500 筆）。另存的 HTML 快照會一併帶著當時的日誌。
  - **清除紀錄需授權且永久稽核**：按「清除紀錄」會**跳出視窗要求輸入人事代號**；未輸入則取消。每次清除都會在日誌上方的**「清除紀錄稽核」區塊**永久留存一筆（人事代號＋時間＋清除筆數），**無法刪除**，並隨 HTML 快照一併保存。
- **儲存／匯入資料** —
  - **儲存**：所有編輯即時存於瀏覽器 `localStorage`；另可用 **匯出 JSON** 下載存檔，日後由「匯入資料」完整還原。
  - **匯入資料**（單一按鈕，**支援一次複選多檔**，自動辨識副檔名）：
    - **JSON／CSV**（本工具匯出的檔案）→ 單檔直接**取代**目前全部場站（可用「重設」還原）。
    - **原始 Excel（.xlsx／.xls）**（一開始提供的「全球場站自動化綜整」與「2026 CUSS 排程」）→ 以**場站代碼合併更新**（相同代碼更新欄位、新代碼新增）。**可一次複選兩個 Excel 檔**，系統會先依場站代碼合併兩檔內容（綜整補現況／系統商／進度、排程補時程分期／啟用日／備註）再套用。專為 EVA 原始格式解析：`● ○ ▲ △ X` 狀態符號、場站代碼後的 `*`（＝經理人 KPI）、狀態後綴系統商、合併儲存格的區域／時程分期均正確辨識；綜整表為現況、排程表為時程，兩者分開處理不互相覆蓋。
    - 匯入前會顯示解析結果並請確認；Excel 解析元件（SheetJS）於匯入時動態載入，需連線；離線時請改用 JSON／CSV。

## 資料來源

初始資料整合自三份原始資料庫（資料基準 2026·05）：

- **全球場站自動化綜整（202605）** — 57 個納管場站的 CUSS／SBD／SBG 現況、系統商、各站進度。
- **2026 CUSS 排程** — 各站建置時程分期與啟用日期。
- **場站航班量／旅客量** — 定位矩陣 X 軸與地圖圓點大小之依據。

共 57 站，CUSS 已上線 18（32%）、SBD 9（16%）、SBG 8（14%）、KPI 場站 10。
所有數字會隨編輯即時重算。世界地圖幾何取自 Natural Earth（110m，公有領域），已簡化並內嵌。

## 技術

原生 HTML／CSS／JavaScript，單一檔案，無外部相依。
唯一的線上資源是「匯出圖片」時才動態載入的 `html2canvas`；離線時可改用匯出 JSON／CSV。
配色採用 EVA Air 設計系統色票。
